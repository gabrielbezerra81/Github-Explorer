<p alig="justify">
Github Explorer é uma aplicação web que permite pesquisar repositórios de usuários e em seguida apresenta informações básicas do repositório. É apresentada uma visão geral do repositório com nome e sua descrição, além de sua quantidade de estrelas, forks e issues abertas. O usuário pode visualizar também quais são as issues abertas, e clicando sobre elas, é redirecionado para a página da issue.
</p>

## Tecnologias utilizadas

* [React JS](https://pt-br.reactjs.org/)
* [Node.js](https://nodejs.org/en/)
* [Typescript](https://www.typescriptlang.org/)

## Demonstração

Veja uma [demonstração](https://github-explorer-gb81.netlify.app/) do app

## Como executar o projeto localmente

### Pré-requisito:

* [Node.js - versão LTS](https://nodejs.org/en/download/)

Instale dependências do projeto:

```
cd github-explorer
'npm install' ou 'yarn'
```

Execute o projeto:

```
'npm start' ou 'yarn start'
```

## Como utilizar

Com a página principal aberta, você deve pesquisar o repositório no formato "autor/repositório". Em seguida, serão listados os repositórios pesquisados e você pode clicar sobre eles e visualizar mais informações.

    formato de pesquisa: 
    
        facebook/react
        gabrielbezerra81/github-explorer
